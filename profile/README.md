Commons OSS is an open-source toolkit for clubs, associations, and nonprofits. Sports clubs, fire departments, hobby groups, charities, community organizations. Built for the DACH region first, written to work anywhere.

## Where to start

- **[commons-oss/commons-oss](https://github.com/commons-oss/commons-oss)** — monorepo, brand assets, design tokens.
- **Attendance tracker** — first product, in design.

## How to contribute

Read [CONTRIBUTING](https://github.com/commons-oss/.github/blob/main/CONTRIBUTING.md) and the [Code of Conduct](https://github.com/commons-oss/.github/blob/main/CODE_OF_CONDUCT.md). Open an issue before sending a large PR.
