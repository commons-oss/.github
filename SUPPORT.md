# Support

Commons OSS is an open-source project maintained by volunteers. Please be patient and respectful when asking for help.

## Where to ask

- **Bug reports** — open an issue in the relevant repo.
- **Feature requests** — open an issue in the relevant repo with the "feature request" template.
- **Questions, ideas, show-and-tell** — [GitHub Discussions](https://github.com/orgs/commons-oss/discussions) (once enabled).
- **Security issues** — see [SECURITY.md](./SECURITY.md).
- **Code of Conduct concerns** — see [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md).

## What to include in a bug report

- What you expected to happen.
- What actually happened.
- Steps to reproduce.
- Environment: OS, runtime version, package versions.
- Logs or screenshots if relevant.

## Response times

This is a volunteer project. Expect responses within a week, sometimes longer. If something is urgent and security-related, use the security channel — those get faster attention.

## Commercial support

For clubs and organizations that need support contracts, custom features, or hosting, see the commercial license at https://github.com/commons-oss/commons-oss.
